Pateros Technological College Official Website 

(Laravel & Vue.Js)
=======

Getting Started
---------------
See the [setup guide](SETUP.md) for a rundown on how to get a development environment up.

Contributing
------------
We welcome all contributions, we recommend you open an issue with your intentions before spending too much time, to ensure no efforts is wasted.

Contributions can be made via pull requests to this repository. If you're unsure of what you can help with, check out the [list of open issues](https://github.com/cMaroon/ptcweb-official/issues).

Seeking Help
------------

### Creating an Issue

If you have something you want to discuss in detail, or have hit an issue which you believe others will also have in deployment or development on the system, [opening an issue](https://github.com/cMaroon/ptcweb-official/issues) is the best way to get help. It creates a permanent resource for others wishing to contribute to conversation. Please **make sure to search first** in case someone else has already addressed the same issue!
